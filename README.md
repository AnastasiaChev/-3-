## Описание

Веб-приложение для управления и просмотра электронных книг с поддержкой фильтрации, администрирования и протоколирования действий, реализованное на HTML, CSS и Python.

---

## Функционал

**Пользовательская часть**  
- Постраничный просмотр всех книг  
- Фильтрация книг по темам («роман», «классика», «учебная литература» и др.)
- Скачивание файлов 
- Отображение обложек и метаданных (название, автор)

**Административная часть**  
- Просмотр каталога файлов с поиском и пагинацией 
- Добавление и удаление книг (с обложкой и метаданными)
- Рейтинг самых скачиваемых книг
- Журнал всех действий пользователей (скачивание, добавление, удаление)
- Защита административных страниц паролем (`admin!!!`)


---

## Технологии

- Все хранится локально (без базы данных)
- Метаданные книг размещаются в файле bool.json
- Статистика скачиваний хранится в файле dowmloads.json
- Лог дейсвтий ведется в файле log.json
- Отладка через VS Code
- Backend: Python (Flask)
- Frontend:  HTML, CSS
- Среда разработки: VS Code
- Используется виртуальное окружение (venv)

---

## Как запустить

1. Скачайте или клонируйте репозиторий.
2. Создайте вирутальное окружение 
3. Активируйте окружение
4. Установите библиотеки, используемые в файле app.py
5. Запустите файл app.py
6. Откройте в браузере появившийся локальный адрес приложения (укахывается в терминале)
7. Для доступа к административным страницам указывается пароль: **`admin!!!`**
